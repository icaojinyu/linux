#### 目标
* 关机/重启

  * `ifconfig`

### 01.关机/重启
|序号|命令|对应英文|作用|
|---|-----|-----|-----|
|01|shutdown 选项 时间| shutdown|关机/重启|

#### 1.1 shutdown
* `shutdown`命令可以安全关闭和重新启动系统

|选项|含义|
|----|----|
|-r|重新启动|

> 提示:
> * 不指定选项和参数,默认表示 ***1分钟*** 之后 关闭电脑
> * *远程维护服务器时,最好不要关闭系统,而应该重新启动系统*

* 常用命令实例:
```bash
# 重新启动操作系统,其中 now 表示现在
shutdown -r now

# 立刻关机,其中 now 表示现在
shutdown now

# 系统在今天的 20:25 会关机
shutdown 20:25

# 系统再过10分钟后自动关机
shutdown +10

# 取消之前指定的关机计划
shutdown -c
```
